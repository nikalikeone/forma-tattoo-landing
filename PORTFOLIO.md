# Материалы кейса FORMA для портфолио

## 1. Короткое описание для GitHub

Адаптивный лендинг частного тату-мастера из Омска в эстетике авангардного арт-журнала. Реализованы галерея работ, блоки доверия, FAQ, форма заявки, SEO-основа и подготовка к статической публикации.

## 2. Описание проекта для портфолио

FORMA — концепт коммерческого лендинга для частного тату-мастера. Проект помогает потенциальному клиенту познакомиться с визуальным стилем мастера, посмотреть работы, понять этапы взаимодействия и отправить исходные данные для консультации.

Основной задачей было совместить выразительный авторский дизайн с понятным пользовательским сценарием. Визуальная система построена на строгой журнальной сетке, асимметричной композиции, крупной типографике и одном кобальтовом акценте. При этом основные действия, навигация и контент остаются доступными на мобильных устройствах.

Проект создан как самостоятельный портфолио-концепт и не выдаётся за клиентскую работу. Демонстрационные изображения и персонаж не принадлежат реальному мастеру.

## 3. Какие задачи решает сайт

- презентует стиль и уровень работ;
- формирует доверие к мастеру;
- объясняет процесс до первого обращения;
- снимает типовые возражения через FAQ;
- собирает информацию для консультации;
- создаёт техническую основу для поискового продвижения в Омске.

## 9. Идеи развития коммерческого продукта

### Этап 1 — реальный лендинг

Заменить концептуальный контент фотографиями и данными мастера, подключить форму, домен, аналитику и инструменты вебмастеров.

### Этап 2 — сайт с поисковыми посадочными страницами

Добавить отдельные страницы направлений: blackwork, fine line, графика, перекрытие и другие реальные услуги мастера.

### Этап 3 — управление контентом

Подключить headless CMS, чтобы мастер самостоятельно публиковал новые работы, зажившие татуировки и свободные эскизы.

### Этап 4 — автоматизация записи

Добавить календарь консультаций, уведомления, учёт предоплаты и интеграцию с CRM или Telegram.
